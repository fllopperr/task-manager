# 3.1 UserFlow

---

## 1. Вход в систему

Старт → Экран входа

Пользователь может:

- Ввести email и пароль

- Перейти к регистрации

Возможные сценарии:

- Успешный вход → Dashboard

- Ошибка входа → Сообщение об ошибке

- Пустые поля → Валидация

---

## 2. Регистрация

Пользователь:

- Вводит email

- Вводит имя

- Вводит пароль

- Подтверждает пароль

Сценарии:

- Успешная регистрация → Dashboard

- Email уже используется → Ошибка

- Пароли не совпадают → Ошибка

---

## 3. Dashboard (Список досок)

Пользователь может:

- Создать новую доску

- Открыть существующую

- Перейти в настройки

- Выйти из аккаунта

---

## 4. Создание доски

Пользователь:

- Нажимает «Новая доска»

- Вводит название

- Подтверждает создание

Сценарии:

- Успешное создание → Доска появляется в списке

- Пустое название → Ошибка

---

## 5. Работа с доской (Kanban)

- На экране доски пользователь может:

- Создать задачу

- Переместить задачу между колонками

- Открыть задачу

- Отфильтровать задачи

- Выполнить поиск

---

## 6. Создание задачи

Пользователь:

- Нажимает «Новая задача»

- Вводит название

- Добавляет описание

- Указывает приоритет

- Выбирает статус

Сценарии:

- Успешное создание → Задача появляется в колонке

- Пустое название → Ошибка

---

## 7. Редактирование задачи

Пользователь может:

- Изменить название

- Изменить описание

- Изменить статус

- Добавить срок

- Добавить метки

- Назначить исполнителя

- Добавить комментарий

- Удалить задачу

---

## 8. Фильтрация и поиск

Пользователь может:

- Отфильтровать по приоритету

- Отфильтровать по исполнителю

- Отфильтровать по метке

- Выполнить поиск по названию

---

## 9. Настройки

Пользователь может:

- Изменить имя

- Изменить email

- Изменить аватар

- Изменить тему

---

## 10. Выход

- Пользователь нажимает «Logout» → возвращается на экран входа.
